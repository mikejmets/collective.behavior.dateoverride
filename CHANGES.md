# Changelog

## 1.0.0 (2025-02-03)

### Features

- Initial release
- Adds `event_date_override` boolean field to events
- Adds `event_date_override_text` text field to events
- Catalog indexes for efficient querying
- Safe indexers for mixed content
- Works with Collections and event listings
- Compatible with Plone 5.2 and 6.0

### Installation

- GenericSetup profile for easy installation
- Catalog upgrade step
- Automatic behavior registration
